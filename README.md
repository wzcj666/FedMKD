# FedMKD
FedMKD: Hybrid Feature Guided Multilayer Fusion  Knowledge Distillation in Heterogeneous  Federated Learning
# Datasets
The datasets used in this paper are CIFAR10, CIFAR100, Tiny-ImageNet, Flowers102, and Skin-Lesions-14. All datasets are publicly available and can be downloaded from open-source repositories on the Internet. The acquisition methods and partial data distribution details are provided in the data section.
# Requirements
1. Python 3.8
2. Pytorch 1.9
3. Some other libraries
